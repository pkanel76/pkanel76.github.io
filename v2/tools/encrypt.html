Encrypt a project

Everything here runs locally in your browser — no key or content leaves this page. Use it to turn any HTML file (from Claude, Replit, anywhere) into an encrypted bundle for the workspace.

1 · Encrypt an HTML project

2 · Add it to the wall

Put the downloaded .json in workspace/projects/, then add an entry to manifest.json:

{ "slug": "…", "title": "…", "source": "Claude", "date": "2026-06", "locker": "A-02" }
3 · Hash a gate passphrase

To change the workspace key, hash the new passphrase and paste the result into GATE_HASH in workspace/index.html.
